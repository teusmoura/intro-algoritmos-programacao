# XX — Título da Atividade

## Enunciado

Descreva aqui o problema que o aluno deve resolver. Seja objetivo e claro.
Use linguagem acessível para iniciantes.

Exemplo: "Leia dois números inteiros e imprima a sua soma."

## Entradas

Descreva cada linha de entrada:

- Linha 1: descrição do dado (tipo e restrição, ex.: inteiro entre 1 e 1000)
- Linha 2: ...

## Saídas

Descreva cada linha de saída:

- Linha 1: descrição do dado a ser impresso

## Exemplos

**Entrada:**
```
(dado de entrada)

**Saída:**
```
(saída esperada)

- Dica 1 sobre a lógica ou sobre a linguagem.

## Erros comuns

- **Erro Y:** descrição e como evitar.

## Como rodar

```bash
# Python
python3 python/solucao.py

# C — compilar e executar
gcc c/solucao.c -o solucao
./solucao
```

Para testar com um arquivo de entrada:

```bash
python3 python/solucao.py < entrada.txt
./solucao < entrada.txt
```
