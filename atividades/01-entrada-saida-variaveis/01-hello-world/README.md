# 01.01 — Hello, World!

Tema: Entrada e Saída / Dificuldade: Fácil

## Enunciado

Escreva um programa que imprime a mensagem clássica "Hello, World!" na tela.

## Entradas

Nenhuma entrada é necessária.

## Saídas

- Linha 1: `Hello, World!`

## Exemplos

### Exemplo 1

**Entrada:**
```
(nenhuma)
```

**Saída:**
```
Hello, World!
```

## Restrições

- Nenhuma entrada é lida.
- O texto deve ser exatamente `Hello, World!` (com vírgula e ponto de exclamação).

## Dicas

- Em Python, use `print("Hello, World!")`.
- Em C, use `printf("Hello, World!\n");`.

## Erros comuns

- **Texto diferente:** certifique-se de usar exatamente `Hello, World!` (maiúsculas, vírgula e `!`).

## Como rodar

```bash
# Python
python3 python/solucao.py

# C
gcc c/solucao.c -o solucao
./solucao
```
